# AvocadoProject


Hello there ,
This is a social app in native Android hope you like it .

## Built With

* Android studio 
* Java
* Kotlin
* MVVM Architecture
* Android jetpack (Room(Sqlite) ,Navigation , Databinging )
* Asp.net restful api
* C#
* Kotlin Coroutines
* Firebase FCM Cloud Messaging
* Firebase real time database
* Microsoft SQL



## Project capabilities
* Mainfeed
* Search
* Search by location
* Profile
* Alerts
* Chat
* Privacy
